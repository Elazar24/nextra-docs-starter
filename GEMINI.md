# Project Overview

This is a Nextra documentation starter project, designed for creating documentation websites. It leverages Next.js for the application framework, Nextra for documentation features, and Pagefind for client-side search functionality. It uses pnpm as the package manager.

# Building and Running

*   **Install Dependencies:** `pnpm i`
*   **Run Development Server:** `pnpm dev`
*   **Build for Production:** `pnpm build`
*   **Start Production Server:** `pnpm start`
*   **Generate Search Index:** `pnpm postbuild` (This runs automatically after `pnpm build`)

# Development Conventions

*   Uses Next.js, Nextra, and Tailwind CSS.
*   TypeScript is used for type checking.
*   Pagefind is integrated for search.
*   Content is written in MDX files, located in the `content/` directory.
